🔍 Client Restriction Architecture

The bow (built by Eduswitch Solutions) uses a layered client-side security architecture. It combines Chromium shell locks (Electron) with native Win32 helper executables to block unauthorized user operations and stream live webcam/screen telemetry.

💻 Technical Code Decomposition

🧪 preload.js (Bridge to Kiosk Container)

As an Electron preloader, this script executes with full access to Node.js APIs before any session page runs in the webview context. It acts as the bridge, injecting secure interfaces into the client web app context.

🔗 Host-IPC Bindings Exposes global helper functions Startsession() and Endsession() on the window scope, passing JSON credentials to the main shell.
⚙️ Process List Leakage Pulls the global list of active operating system processes from the main process and stores them directly in localStorage for session portals to inspect.
📡 Message Interceptor Listens to HTML message events in the browser frame (e.g. from Talview and Eduquity iframe scripts) and tunnels opentab/stop codes to the host shell.

🖥️ chrome-tabs/renderer.js (WebRTC & Telemetry Streamer)

A heavy, obfuscated client script loaded directly by the browser container. Governs media capturing, audio monitoring, WebRTC server channels, and records desktop screens.

📡 WebRTC Web sockets Negotiates connections using remote Ice Servers, establishing channels under webrtc/users/ on Firebase Realtime Database.
📹 Media Stream Recorder Sets up parallel webcam and screen recorders, capturing 30s clips, encrypting them via CryptoJS (ECB padding), and uploading chunks to remote storage.
👁️ tensorflow.js monitoring Initializes webworkers running BlazeFace. Calculates distance between pupils, flags face count violations, cell phone objects, and writes flags.

⚙️ Native Win32 monitoring Helpers

Because Electron javascript runs inside a Chromium sandbox, the browser spawns native C++ / C# (.NET) executables to enforce deep operating system hooks and gather hardware data.

🔒 restrictions-diablewinkey-winformsapp.exe Loads a low-level keyboard hook (WH_KEYBOARD_LL) on Windows. Intercepts key codes and discards keypress events for the Windows key, Alt+Tab, and Ctrl+Esc.
🖥️ detectvirtualdesktop/virtualdesktop11.exe Intercepts requests to the Windows Virtual Desktop API (IVirtualDesktopManager). Shuts down the browser or alerts monitors if a user tries to switch tabs.
🔍 vmdetect.exe Queries kernel flags and hypervisor indicators (e.g. Cpuid leaf 0x40000000). Inspects BIOS strings for names like "VMware", "VBOX", "QEMU" or "Parallels".
